<br>

## 프로젝트 소개
이번학기 데이터베이스 프로그래밍 수업을 수강하면서 배운 데이터베이스 프로그래밍 기법을 사용한 수강신청 프로그램입니다.

Oracle 데이터베이스와 Tomcat 서버를 사용하였습니다. 

주요기능으로는 다음와 같습니다. 

- 로그인
- 사용자 정보 수정 
- 수강신청
- 수강조회
- 수강신청 취소
- 강의목록 검색
